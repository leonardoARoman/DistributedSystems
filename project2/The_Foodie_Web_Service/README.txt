1. To run the application run the app.py file
>> python app.py
2. To make http requests open a new terminal and run curl commands
>> curl http://127.0.0.1:5000/restaurant/washington+eve+pleasantville+nj+08232
3. Also you can make http requests using postman
4. The application is running on local host port 5000
5. Go to http://localhost:5000 to see tentative webapp
6. CREDIT: http://blog.luisrei.com/articles/flaskrest.html
